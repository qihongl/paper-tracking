LLM-Memory
Exploratory and Assimilating Reflection: Reflective Recall Cycle for Long-term Memory
Senrayan G, Yamada M, Jindal I, Purohit K et al. · arXiv cs.AI, 2026-07-20 (preprint) LLM-Memory Cross-cutting

Approach: LLM-based autonomous agents are stateless with limited context windows, so they need external memory for long-term interaction and dynamic knowledge reasoning. The authors propose Exploratory-Assimilating Reflection (EAR), which pairs Exploratory Reflection (iterative search that bootstraps retrieval and collects candidate memories) with Assimilating Reflection (compressing and integrating retrieved items into a coherent working set). It is evaluated on long-horizon agent benchmarks requiring heterogeneous memory stores.

Finding: EAR delivers high initial retrieval performance and sample-efficient adaptation relative to prior memory-retrieval methods, and handles mixed heterogeneous memory better. No effect sizes are reported in the abstract, but the architecture is directly relevant to Pillar 1 (controlled, attention-based episodic memory in transformers/agents).

⚠ Preprint; benchmark names and absolute gains not stated in the abstract — verify against the full paper before citing.
The Art of Not Forgetting
Atmuri A, Kumar A, Bhogarajula YR · arXiv cs.AI, 2026-07-20 (preprint) LLM-Memory KV-Networks

Approach: Introduces CMP (Cognitive Memory Primitive), an architecture that represents inputs as sparse relational codes stored in a two-tier competitive memory and learns entirely through local, gradient-free updates — no backpropagation anywhere. It is used to test a specific hypothesis: that catastrophic forgetting, usually treated as a training-time defect patched with replay or regularization, is instead a structural consequence of how backpropagation assigns credit.

Finding: A learning rule that is local and sparse by construction resists catastrophic forgetting without replay or regularization, reframing forgetting as an architectural rather than optimization failure. This dovetails with complementary-learning-systems thinking and is relevant to Pillars 1 and 3 (memory encoding and key-value-style storage).

⚠ Preprint; demonstration is computational/theoretical — no large-scale LM validation yet.
Retain or Consolidate? Budget-Dependent Operator Selection for Language Agent Memory
Kang Q, Liu M, Kai S, Liang K, Tang Z et al. · arXiv cs.AI, 2026-07-20 (preprint) LLM-Memory

Approach: Language agents depend on memory across interactions, but limited context windows and inference cost cap how much can be used at once. The paper formalizes the trade-off between retention (keeping raw records — exact but budget-heavy) and consolidation (compressing/combining records — better coverage per token but risking loss of query-critical detail), and proposes budget-dependent operator selection between the two.

Finding: Neither retention nor consolidation is universally preferable; the optimal strategy depends on the memory budget and the query. The work frames agent memory explicitly in consolidation terminology, which maps cleanly onto Pillar 1's lingering/episodic memory framing.

⚠ Preprint; the abstract reports the qualitative trade-off but no quantitative effect sizes.
Persistent Sparse Autoencoders: Learning Feature Timescales in Language Models
Luo H, Espinosa Zarlenga M, Jamnik M · arXiv cs.CL, 2026-07-19 (preprint) LLM-Memory Cross-cutting

Approach: Standard sparse autoencoders (SAEs) encode each token independently and discard information that persists across a sequence. The authors extend SAEs with a per-feature persistence coefficient that learns which features should persist and for how long, then evaluate on language-model activations.

Finding: Persistent SAEs keep competitive reconstruction while learning a spectrum of feature timescales: fast features behave as locally interpretable detectors, slower features integrate across the sequence. This offers a mechanistic-interpretability window onto memory-like persistence in LLMs — a potentially useful probe for the lingering-memory project.

⚠ Preprint; persistence is learned post hoc on activations, not causal.
Schema-Episodic
Selective convergence and graded divergence of hippocampal and amygdala subregions using functional connectivity
Erigüc DY, Marsiglia M, John A, Bayrak S, Jakovcic A, DeKraker J et al. · bioRxiv, posted 2026-07-18 (preprint; orig. sub. 2026-07-06) Schema-Episodic Cross-cutting

Approach: Using resting-state fMRI from 722 Human Connectome Project Young Adult participants, the authors mapped hippocampal and amygdalar subregions within a single cortex-wide functional-connectivity framework, quantifying how each MTL subregion is embedded into distributed isocortical systems.

Finding: Hippocampal and amygdalar subregions show selective convergence (shared cortical targets) and graded divergence (subregion-specific embeddings), linking memory and affect systems along the medial-temporal axis. This is the only clearly Pillar-2-relevant item in the window and is a high-N (n=722) systems-mapping paper, though correlational.

⚠ Preprint; resting-state correlational design with no causal/manipulation — treat the convergence/divergence claims as descriptive.
KV-Networks
C²KV: Compressed and Composable KV Cache Reuse for Efficient LLM Inference
Du C, Chen J, Tang H, Liu K, Lan T et al. · arXiv cs.CL, 2026-07-20 (preprint) KV-Networks

Approach: Long-context LLM serving is bottlenecked by storing and accessing large KV caches. Existing KV-reuse methods cut prefill compute but ignore the storage/access cost; naive combination with compression often degrades. C²KV combines KV compression with non-prefix (composable) KV reuse so both costs fall together.

Finding: The method enables compressed and composable KV-cache reuse, reducing the storage/access bottleneck while preserving the compute savings of reuse. This is squarely on Pillar 3 (KV memory networks) — it addresses how structured KV state is stored and reused under long contexts, alongside the position/event memory questions.

⚠ Preprint; engineering-focused with no mechanistic/behavioral claim — included for direct Pillar-3 relevance.
HyMCache: A KV Cache Framework for Multi-Turn LLM Serving with CXL-Hybrid Memory
Jang H, Song I, Noh SH, Kim J · arXiv cs.DC/cs.LG, 2026-07-20 (preprint) KV-Networks

Approach: Long-context, multi-turn, and agentic workloads increasingly reuse previously processed context, making KV-cache reuse essential. HyMCache integrates CXL-hybrid memory (small in-device DRAM + large SSD behind a CXL interface) to serve TB-scale reusable KV states, exploiting the read-dominant, predictable, append-only nature of multi-turn KV access. Evaluated on a real CXL-HM prototype in single-aggregator and PD-disaggregated serving.

Finding: Under matched DRAM budget, HyMCache outperforms local LMCache by 3.0× (single-node) and 1.45× (PD-disaggregated); versus 1 TB distributed-DRAM Mooncake it incurs ~30% lower performance at 16× less DRAM. A systems-level contribution to scaling Pillar-3 KV memory to cluster scale.

⚠ Preprint; systems/engineering result, not a memory-mechanism claim.
Regularize or Localize: When Training-Time KV-Cache Geometry Pays Under Quantization
Sun L, Harn P-W, Zhang Z, He P, Qin X · arXiv cs.LG, 2026-07-19 (preprint) KV-Networks Cross-cutting

Approach: Tests whether σreg — LeJEPA's anti-collapse objective — reshapes representations during standard autoregressive LM pretraining and whether the resulting geometry helps KV-cache quantization. They train 110M-parameter models on 10B FineWeb tokens.

Finding: At λ=0.01, σreg reduces hidden-state pairwise-cosine anisotropy by 38% with <0.35% perplexity cost, but the change does not propagate from hidden states to the KV cache. Quantization gains require localized (per-head/layer) geometry, not just global regularization — a useful constraint for Pillar-3 KV-compression work.

⚠ Preprint; modest scale (110M) — check whether the anisotropy→KV gap persists at frontier scale.
Encoding-Retrieval
Redundant prefrontal hemispheres adapt storage strategy to working memory demands
Tschiersch M, Umakantha A, Williamson RC, Smith MA, Barbosa J, Compte A · Nature Communications, 2026-07-20 Encoding-Retrieval

Approach: Simultaneous bilateral prefrontal cortex recordings from three male macaque monkeys performing a visuo-spatial working-memory task, combined with attractor-network simulations and validation in human and monkey data. Tests two opposing theories: a specialized contralateral architecture vs. a redundant fail-safe one.

Finding: Both hemispheres predicted behavioral imprecision equally; decoding errors were only weakly correlated between hemispheres; and serial dependence stayed local within hemispheres — a redundant, weakly coupled organization. Attractor simulations showed redundancy improved precision when demands were below capacity, while weak interhemispheric coupling raised capacity under high demand via hemispheric specialization, reconciling the two theories.

⚠ n=3 monkeys; the human validation mitigates but does not replace the small N.
Working memory deficits in hippocampal amnesia are associated with apathy
Attaallah B, Maio MR, Tabi YA, Nobis L, Husain M et al. · Journal of Neuroscience, 2026-07-20 (online ahead of print) Encoding-Retrieval

Approach: 27 patients with LGI1-antibody limbic encephalitis (predominant hippocampal involvement) vs. 27 age- and gender-matched healthy controls, tested on an object-location continuous-report working-memory task analyzed with Bayesian mixture modeling.

Finding: Patients showed predominantly elevated misbinding errors — incorrectly associating objects with non-target locations — with unaltered guessing and only marginally reduced precision. Critically, apathy was positively associated with misbinding exclusively in patients, independent of depression, suggesting hippocampal pathology degrades WM binding and that apathy mediates part of the deficit.

⚠ Clinical cross-sectional sample (n=27, 21M/6F); causal direction of the apathy link is correlational.
Beyond memory capacity: A probabilistic, dual store model of visuospatial working memory
Aponte EA, Perumal TM, Cormack F, Chatham CH · PLoS Computational Biology, 2026-07-20 Encoding-Retrieval Cross-cutting

Approach: A computational model of visuospatial working-memory dynamics that extends classical resource/stick ideas, fit to data from three clinical trials (neurotypicals and autistic participants on a smartphone-based sequential VSWM task) and a large prodromal-Alzheimer's trial.

Finding: Visual information in the sequential task is stored in independent pools of contrasting resources with perfect and imperfect retrieval rates, capturing how items are stored, retained, and removed as new information arrives. The paper argues computational models plus remote cognitive testing are now mature for large-scale clinical WM research — a useful modeling scaffold for Pillar-2/3 memory dynamics.

⚠ Model is fit to a smartphone sequential task; generalizability to classic WM paradigms is untested.
