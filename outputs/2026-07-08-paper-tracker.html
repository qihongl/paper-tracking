LLM-Memory
Memory in the Loop: In-Process Retrieval as Extended Working Memory for Language Agents
Yusuf Khan, Carlo Lipizzi (Stevens Institute of Technology) · arXiv:2607.05690, 6 Jul 2026 · Preprint LLM-Memory

Approach: Moves memory inside the LLM agent's observe-reason-act loop, reading and writing on every step. Replaces networked stores (~110ms latency) with an in-process store (~100µs). Invokes the extended-mind thesis: a store fast enough to be constantly and directly available becomes extended working memory, not a tool the agent merely consults. Tests across four GPT-5-class models under a bounded window recall task.

Finding: In-loop memory improves recall from 0/5 to 3.6–4.8/5. Redundant actions rise monotonically with latency: 0.0 of 12 at in-process speed vs. 7.2 of 12 at 110ms cloud round trip (exact permutation p=0.0079). Store never lost a fact (244/244 writes kept); every miss traces to the agent's read policy, not the store. The dominant per-step cost is embedding (~200–400ms); pairing with a small local embedder drops the complete operation to ~40µs. Mechanistically, this is a direct demonstration that memory access latency causally determines effective working memory capacity in LLM agents.

MemDefrag: Latent Memory Defragmentation for Large Language Models
Ruiyi Yan, Zhuoyuan Mao, Yiwen Guo · arXiv:2607.05969, 7 Jul 2026 · Preprint LLM-Memory

Approach: Diagnoses performance degradation in latent memory LLMs (MemoryLLM, M+) during memory updates. Probes layer-wise attention density to discover that middle transformer layers concentrate the highest attention on target memory fragments — an inherent tracing signal. Proposes MemDefrag, a training-free framework using this signal to rank, reorder, and filter stored latent memories, plus an informativeness-guided proportional forgetting mechanism.

Finding: MemDefrag achieves 43.0% knowledge retention after 50 memory updates vs. 17.4% (MemoryLLM) and 17.6% (M+). Generalizes across LLM architectures and latent-memory variants. The key mechanistic insight: positional encoding misalignment degrades memory update quality, and a middle-layer attention-density tracing signal can correct this without retraining — a lightweight fix for a fundamental architectural problem in latent-memory LLMs.

Preprint; no ablation on whether the tracing signal degrades with model scale.
NapMem: From Passive Retrieval to Active Memory Navigation — Learning to Use Memory as a Structured Action Space
Yue Xu, Yutao Sun, Yihao Liu, Mengyu Zhou et al. · arXiv:2607.05794, 7 Jul 2026 · Preprint LLM-Memory

Approach: Frames long-term user memory as a structured action space rather than passive retrieval. Builds a linked multi-granularity memory pyramid: raw conversations → typed memory records → topic tracks → user profiles, connected through provenance relations. The agent is trained via memory-tool reinforcement learning to navigate these granularity levels, selecting the appropriate memory resolution for each query.

Finding: NapMem is competitive on PersonaMem-v2, LongMemEval, and LoCoMo while largely preserving general reasoning and tool-use abilities. Ablations show that both structured storage and the learned navigation policy contribute independently. The key architectural claim: treating memory granularity as an action-space decision — not a fixed retrieval parameter — improves the quality of memory use in personalized agents.

Akashic: A Low-Overhead LLM Inference Service with MemAttention
Yang Liu, Zhaokai Luo, Huayi Jin, Ruozhou He et al. (Junhao Hu) · arXiv:2607.05708, 7 Jul 2026 · Preprint LLM-Memory KV-Networks

Approach: Proposes MemAttention, which organizes accumulated agent context into bounded chunks and models cross-chunk semantic relationships. Combines this with hardware-software co-designed memory placement that co-locates likely co-retrieved chunks, reducing retrieval fragmentation and I/O overhead. Contrasts with replay-the-full-history baselines.

Finding: Across four workloads and three model sizes, Akashic improves task accuracy by up to 10.2 points, throughput by up to 1.21×, and sustainable request rate by up to 1.88× over prior memory baselines. The cross-chunk semantic modeling is the key innovation — it preserves evidence that would otherwise be lost when chunking the context.

LongCrafter: Towards Diverse Long-Context Understanding via Evidence-Graph-Guided Instruction Synthesis
Chenhao Yuan, Yinhao Xu, Shuwen Xu et al. (Pengfei Cao, Jun Zhao, Kang Liu, CAS) · arXiv:2607.06160, 7 Jul 2026 · Preprint LLM-Memory

Approach: Builds a hierarchical task taxonomy with 32 fine-grained long-context task types spanning local/shallow to global/deep understanding. Constructs task-aligned long contexts, decomposes them into explicit evidence graphs modeling cross-paragraph dependencies, and generates instruction-response pairs grounded in located evidence spans.

Finding: Models fine-tuned on LongCrafter data outperform all SFT baselines and even the official post-trained models on LongBench, LongBench v2, and LooGLE (Qwen2.5-7B and LLaMA-3.1-8B). Largest gains on high-difficulty tasks. Trained models locate evidence robustly regardless of position, effectively mitigating the "lost in the middle" problem. Mechanistically interesting because the evidence-graph grounding enforces faithful, traceable reasoning about long-context dependencies.

KV-Networks
FreqDepthKV: Frequency-Guided Depth Sharing for Robust KV Cache Compression in Long-Context LLM Inference
Anna Córdoba, Adam Puente Tercero, Nerea Angulo Hijo et al. · arXiv:2607.06519, 7 Jul 2026 · Preprint KV-Networks

Approach: Factorizes adjacent-layer KV states into shared low-frequency depth components and sparse high-frequency residuals. A lightweight online probe assigns each attention head to shared-depth, residual-depth, or exact cache mode based on its contribution to reconstruction-sensitive attention logits, adapting compression policy to prompt structure without retraining.

Finding: At 32K-token prefill: 58.3 Exact Match, 63.0 F1, 32.5 ROUGE-L, 48.1 pass@1 — closely matching full KV cache. Achieves 70.4 tokens/s decoding throughput, 2.06s TTFT, 6.2 GB peak KV memory, for a 3.9× effective compression ratio. The frequency-guided factorization is the key insight: low-frequency components are shared across layers (highly compressible), while high-frequency residuals capture the layer-specific evidence needed for retrieval and reasoning.

Preprint; companion paper DepthWeave-KV (2607.06523) by same group achieves 8.3× compression with token-adaptive routing — included in excluded section.
Encoding-Retrieval
The Neural Signature of Simple Overlearned Temporal Expectations Resembles Episodic Retrieval
Linda Sempf, Peter Vavra, Toemme Noesselt (Otto-von-Guericke University Magdeburg) · bioRxiv, 7 Jul 2026 · Preprint Encoding-Retrieval

Approach: fMRI study using a multi-day detection paradigm with auditory cues preceding visual targets at likely vs. unlikely time points. Compares trained participants against untrained controls to isolate overlearned temporal expectations while controlling for parallel learning processes. Uses Neurosynth association tests to characterize the functional profiles of activated regions.

Finding: After training, left inferior parietal cortex, retrosplenial cortex, and prefrontal cortex were engaged in processing overlearned temporal expectations. Critically, the trained group showed inverted brain responses as a function of stimulus likelihood compared to controls. Neurosynth meta-analysis revealed that these regions have been previously implicated in episodic retrieval and simulation. The authors propose that frequently used temporal expectations may, over time, recruit episodic memory routines to simulate likely future events. Mechanistically, this suggests a convergence of temporal prediction and episodic retrieval circuitry — temporal context becomes so overlearned that it triggers retrieval-like neural states.

Cross-cutting
Narrative World Model: Narratology-Grounded Writer Memory for Long-Form Fiction
Mohammad Saifullah, Thomas Kornmaier, Taaha Kazi, Vasu Sharma et al. · arXiv:2607.05577, 6 Jul 2026 · Preprint Cross-cutting

Approach: Builds a writer-memory system pairing a typed temporal-state graph grounded in narratology (who knows what, when they learned it, whether events precede narration, how relationships shift) with query-conditioned hybrid retrieval. Evaluates via a held-constant Opus 4.8 reader over only chapter-safe evidence, comparing against Graphiti/Zep (Rasmussen et al., 2025) and GraphRAG on a validated multi-hop narratological QA benchmark.

Finding: NWM substantially and significantly outperforms Graphiti/Zep on multi-hop narratological QA. The advantage is representational, not an artifact of extraction quality: it survives rebuilding the baseline with NWM's own extractor. The typed temporal-state graph structure — which tracks entities, events, and their temporal/doxastic relations — is what drives the gain. Relevant to Pillar 3 (KV-Networks) because the temporal-state graph is effectively a key-value memory for narrative state, and to Pillar 1 (LLM-Memory) because it demonstrates how structured temporal representations dramatically improve memory retrieval over flat approaches.

Application domain (fiction writing) is far from the researcher's focus; included for the temporal-state graph mechanism.
Peripheral
Self-Review Reinforcement Learning with Cross-Episode Memory and Policy Distillation
Muhammad Zain Amin, Kibele Sebnem Yildirim · arXiv:2607.05541, 6 Jul 2026 · Preprint Peripheral

Approach: Introduces SRRL, a training framework that embeds explicit self-review into each RL episode. When a first-pass LLM response fails, the model generates a self-review identifying what went wrong, conditions an improved second attempt, and internalizes improvements into the base policy via selective distillation. A cross-episode memory buffer retains successful self-reviews for reuse in future episodes.

Finding: SRRL consistently outperforms the RLVR baseline (GRPO optimizer) on GSM8K across Qwen 3-4B and OLMo-3-7B, with greater learning efficiency. The cross-episode memory mechanism — retaining and reusing successful self-reviews — is what makes this relevant: it's a simple but effective episodic memory for RL-trained LLMs that explicitly addresses the credit-assignment problem in sparse-reward environments.
