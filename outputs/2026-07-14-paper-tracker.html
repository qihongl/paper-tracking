Can a Language Model Learn Facts Continually in Its Weights?
Charles O'Neill · arXiv, 13 Jul 2026 LLM-Memory

Approach: Follows invented facts written into Qwen3 model weights from creation through sequences of up to 100 later writes, using held-out questions of five types. Compares bare-statement training against diverse-restatement training, and systematically measures retention, interference, and the recitation-to-use gap across sequential writes.

Finding: Facts can be behaviourally forgotten without being erased — forgotten facts retain most of the log-probability added by their write, and 70% of wrong answers about them contain the most recently written fact. Diverse restatements reduce the recitation-to-use gap from 27.4 to 5.4 percentage points. After 20 sequential writes, bare-statement facts retain 1% accuracy while broad-study facts retain 46%. Later writes cause interference regardless of how the earlier fact was stored. When facts must survive later writes, context (not weights) is the reliable channel. Mechanistically, this demonstrates that LLM weight storage behaves like a question-keyed associative memory where later writes redirect the retrieval pathway.

Relevance: Directly speaks to Pillar 1 (LLM lingering memory) — the question-keyed overwriting mechanism is a transformer analogue of proactive interference in episodic memory. The finding that context is more reliable than weights for composed facts parallels the hippocampal-cortical distinction in CLS theory.

MLPs are Hebbians: Constructing Efficient Fact-Storing MLPs for Transformers
Roberto Garcia, Jerry Liu, Ronny Junkins, Sabri Eyuboglu, Atri Rudra, Christopher Ré (Stanford) · arXiv, 10 Jul 2026 LLM-Memory KV-Networks

Approach: Develops a closed-form Hebbian construction for Transformer MLP layers that stores facts at information-theoretically optimal rates. The construction analyses the decoding margin of MLPs — how cleanly a stored fact's output separates from noise — rather than just storage capacity. Validated against prior constructive models (Meng et al. ROME, Geva et al.) and mechanistic interpretability results.

Finding: First Transformer-compatible construction achieving all three empirically observed properties: (i) optimal fact storage scaling under isotropic embeddings, (ii) handling of arbitrary input/output geometries, and (iii) functioning inside full Transformer blocks for factual recall. Uses 10–104× fewer parameters than prior constructions at matched fact count. Enables modular fact editing by directly swapping an MLP layer — a clean demonstration that fact storage in LLMs follows associative memory principles with Hebbian-like weight updates.

Relevance: Bridges Pillar 1 and Pillar 3 — the Hebbian construction provides a normative account of how LLMs implement associative (key-value) memory in their MLP layers. The modular editing capability is a proof-of-concept for targeted memory modification without full retraining.

One Mechanism for Many Mental Spaces: A Shared Router over a Value Slot in Language Models
Oliver Steele, Jiangtao Wen, Yuxing Han · arXiv, 11 Jul 2026 LLM-Memory Cross-cutting

Approach: Tests whether transformers implement Fauconnier's mental-space theory (a unified space-building operation) or formal semantics (separate logics for modal, temporal, doxastic spaces). Uses Distributed Alignment Search to train subspaces that control one space type (counterfactual, belief, fictional, temporal, or memory) and tests cross-type generalisation across three model families. Mechanistically localises the router to a low-rank subspace acting through a few late-layer attention heads.

Finding: The model uses one router/slot format across the entire inventory: a reusable value slot stores attributed content, and a causally manipulable router (the space index) selects which space is read. A subspace trained on one space type also controls others well above random. The router is low-rank, composes additively with entity identity, and drives inference. Composing space-builders mints a fresh router over the shared slot — a mechanistic implementation of mental space nesting. Memory is one of the spaces in this unified inventory.

Relevance: Cross-cutting for Pillar 1 — provides a mechanistic account of how LLMs represent the distinction between actual and remembered content, with direct implications for how episodic memory could be implemented as a space-building operation in transformer architectures.

Structured Thoughts for Improved Reasoning and Context Pruning
Zain Sarwar, Supriyo Chakraborty, Berkcan Kapusuzoglu, Chia-Hsuan Lee, Anirban Das, Stephen Rawls, Kartik Balasubramaniam, Sambit Sahu (JPMorgan Chase) · arXiv, 11 Jul 2026 LLM-Memory

Approach: Introduces a framework that organises LLM chain-of-thought reasoning into alternating <try> (exploratory scratch work) and <outcome> (distilled conclusion) blocks. Constructs a dataset by segmenting reasoning traces and prompting an LLM to summarise each step. Fine-tunes pretrained models on this reformatted data, then implements context pruning by discarding <try> blocks after each step.

Finding: Fine-tuning on structured thoughts yields up to 8.08% improvement on reasoning benchmarks compared to standard SFT. The pruning mechanism achieves ~85% memory/context savings with an 8.67% performance drop on mathematical tasks. This is a form of gist-based memory consolidation: exploratory computation is jettisoned, but its conclusions are retained.

Relevance: The try/outcome pruning paradigm maps onto the distinction between encoding (detailed processing) and consolidation (gist extraction) in episodic memory. For Pillar 1, this suggests a concrete mechanism for implementing selective forgetting of intermediate representations while preserving conclusions.

⚠ Preprint; pruning performance measured only on mathematical tasks — generalisability to open-ended reasoning unclear.

Emergent Generalization by Representation Learning in Artificial Neural Networks
Hardik Rajpal, Dan Goodman (Imperial College London) · arXiv q-bio.NC, 11 Jul 2026 Schema-Episodic Cross-cutting

Approach: Trains RNNs with an explicit information bottleneck on a time-series prediction task and tracks causal emergence (an information-theoretic measure of macroscale structure) across the memorisation-to-generalisation transition. Then analyses CA1 hippocampal population activity from mice learning an alternating maze task, applying the same causal emergence metric to neural data.

Finding: The information bottleneck is necessary for rotational and out-of-distribution generalisation in RNNs. Causal emergence follows a non-monotonic trajectory — initial decrease, minimum, then rise to maximum — even as prediction loss falls monotonically. This trajectory scales with task complexity and predicts generalisation performance. Strikingly, CA1 hippocampal activity in maze-learning mice shows analogous non-monotonic emergence dynamics that track behavioural performance. Low-dimensional, emergent representations confer a functional advantage for generalisation, supporting a causal role for learned neural manifolds in cognition.

Relevance: Directly bridges Pillar 2 (hippocampal mechanisms) and computational modelling. The non-monotonic emergence dynamics provide a quantitative signature that could be used to track schema formation. For the Schema-Guided Episodic Memory RNN project, the causal emergence metric offers a principled way to measure when representations transition from memorised to generalised.

⚠ Preprint; CA1 analysis is on a single maze task — replication across paradigms needed to establish generality of the non-monotonic signature.
KV-Networks
Remembering Distinct Items, Not Tokens: A Learnable Dirichlet-Process Cache Between State-Space Models and Attention
Siddharth Pal, Viktoria Rojkova · arXiv cs.LG, 10 Jul 2026 KV-Networks

Approach: Proposes a sparse KV cache that allocates a slot only when an input is novel, using the DP-means clustering rule (small-variance limit of a Dirichlet-process mixture) as the key-value memory operator. Implements two variants: static cache with fixed concentration, and surprise-adaptive with concentration following the recent novelty rate. Tests on synthetic associative-recall benchmarks with controlled redundancy, plus four real-world event streams (recommendation, systems logs, clinical events, insurance claims).

Finding: The DP cache matches full-attention recall while storing only distinct items, dominating every fixed-budget eviction cache on the recall-versus-size frontier. On a state-space backbone, it answers both recall and long-range aggregate queries at the lowest memory of any tested model. Critically, a two-parameter novelty-threshold gate trained end-to-end on task loss recovers the DP-means rule exactly, while an over-parameterised gate fails — the inductive bias of the DP prior, not capacity, drives the result. Memory scales with distinct information, not token count.

Relevance: Core Pillar 3: directly implements a key-value memory where the write policy is governed by a Dirichlet-process prior — a Bayesian nonparametric model of how distinct events are accumulated. This is a formal bridge between nonparametric Bayesian models of human memory (e.g., Anderson's rational analysis) and transformer KV caching. Companion paper (2607.10441) extends this to a full tripartite working-memory architecture.

Context by Distinct Information: An Auditable Dirichlet-Process Working Memory for Long, Redundant Context Streams
Siddharth Pal, Viktoria Rojkova · arXiv cs.LG, 11 Jul 2026 KV-Networks

Approach: Companion study to 2607.09889. Develops the allocate-on-novelty DP cache as a working-memory component within a tripartite architecture: recall-carried information goes to a content-addressed novelty cache, summary-carried information to a recurrent state, and locality-carried information to a recency window. Tests on character-level control tasks and next-code prediction over synthetic Medicare claims at thousand-event horizons.

Finding: Novelty-gated attention reaches full-attention performance while attending to ~50% of tokens. The coupled DP cache + state-space summary matches full-attention coupling at reduced cost, with advantage growing as context lengthens. On thousand-event horizons, the coupled component leads full attention and every fixed-budget eviction policy. The retained memory is an inspectable table of templates, codes, drugs, or places — not an opaque state vector. This demonstrates that context can scale with distinct information rather than tokens, in a working memory that is content-addressable and auditable.

Relevance: Pillar 3: the tripartite architecture (novelty cache / summary state / recency window) maps directly onto the dissociation between item-specific (episodic), gist-based (semantic), and recent (working memory buffer) components in human memory. The auditable memory table is especially interesting as a testbed for the KV memory network framework.

From Self-Attention to Connection Laplacian: A Unified Operator View of Transformers
Binbin Lin, Wei Chen, Yalun Li, Wenxiao Wang, Jieping Ye, Xiaofei He (Zhejiang University) · arXiv cs.LG, 12 Jul 2026 KV-Networks Cross-cutting

Approach: Reformulates self-attention as a connection walk over a token-position graph: messages are aggregated by a nonnegative walk matrix while being transported along each edge by a learned linear map. Proves that single-head attention is exactly a connection propagation step with constant transport, and multi-head attention is a single edge-dependent connection walk. Analyses trained transformers from 124M to 8B parameters for geometric structure in their attention operators.

Finding: Effective attention graphs converge to stable geometric operators in deeper layers. Learned transports self-organise into approximate scaled isometries, and both phenomena strengthen consistently with model scale. The framework clarifies when the generator reduces to a random-walk connection Laplacian (stochastic, reversible, metric-compatible transports). Provides operator-level tools for analysing transformer attention as a geometric diffusion process.

Relevance: Pillar 3: provides a rigorous geometric foundation for understanding KV attention as a diffusion process on a structured graph. The connection-Laplacian formulation may offer new ways to analyse interference patterns in KV memory networks — interference corresponds to distortions in the transport operator.

Infrared Organization and Critical Cognitive Field Formation in Transformer Dynamics
Byung Gyu Chae · arXiv cs.LG, 12 Jul 2026 Cross-cutting LLM-Memory

Approach: Extracts relaxation spectra directly from Transformer layer Jacobians across training, network depth, and model scale using public Pythia checkpoints. Measures the time-scale density of states (TDOS), memory self-energy, cognitive forgetting gap, memory kernel, and infrared critical exponent — concepts from Cognitive Field Theory applied to neural network dynamics.

Finding: Training produces progressive infrared accumulation of slow relaxation modes, yielding an approximately flat TDOS with ρ(λ)~λ⁻⁰·¹ and scale-free memory kernels K(t)~t⁻¹. The memory self-energy exhibits a pronounced transient maximum during early optimisation before relaxing toward a metastable near-critical regime. This near-critical state corresponds to the smallest cognitive forgetting gap and largest collective susceptibility. The same dynamical behaviour reproduces across training, depth, and scale — suggesting infrared slow-mode organisation as a universal collective principle of Transformer dynamics.

Relevance: Cross-cutting: the concept of a "forgetting gap" and scale-free memory kernel provides a formal bridge between transformer dynamics and power-law forgetting in human episodic memory (cf. Wixted & Carpenter, 2007). The infrared critical exponent may serve as a quantitative signature of memory regime — relevant to both Pillar 1 (LLM lingering memory) and Pillar 2 (hippocampal consolidation dynamics).

⚠ Preprint; single author, Cognitive Field Theory is a novel framework not yet widely validated. Measurements on Pythia only; broader model coverage needed.
Feedback-Coupled Memory Systems in Continuous Time
Stefano Grassi · arXiv cs.AI, 24 Jun 2026 Cross-cutting

Approach: Formalises the Feedback-Coupled Memory Systems (FCMS) architecture in continuous time by defining the agent update operator through Mechanism-Based Intelligence (decentralised price mechanism) and the environmental update operator through a Coupled Memory Graph Process (non-Markovian substrate that records trajectory history). Analyses Lyapunov global dissipativity conditions and validates with numerical simulation (N=2) and mean-field (N=10⁶).

Finding: Derives a computable Lyapunov global dissipativity threshold: 4β²<2ημγ², generalising both the discrete FCMS stability condition (4ηβ²<γ) and the CMGP physical bifurcation threshold (α_c=1/K). The universal organising principle: memory dissipation must outpace feedback gain. When the threshold is violated, a self-reinforcing coordination cascade emerges. This provides a formal stability criterion for attractor-based memory systems with feedback.

Relevance: Cross-cutting: the dissipation-vs-gain tradeoff is a formal analogue of the stability-plasticity dilemma in neural memory systems (cf. Grossberg, 1980). The Lyapunov analysis provides a rigorous theoretical framework for understanding when attractor memory networks transition from stable storage to runaway interference — relevant to both hippocampal attractor models and transformer KV cache dynamics.
