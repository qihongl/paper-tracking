LLM-Memory
Synapse: Empowering LLM Agents with Episodic-Semantic Memory via Spreading Activation
Jiang et al. (Tianming Liu, University of Georgia) · ACL 2026 Findings, July 2026 LLM-Memory

Approach: Introduces Synapse, a unified memory architecture that models agent memory as a dynamic graph where relevance emerges from spreading activation rather than static vector similarity. Integrates lateral inhibition and temporal decay to dynamically highlight relevant sub-graphs while suppressing interference. Uses a Triple Hybrid Retrieval strategy fusing geometric embeddings with activation-based graph traversal. Evaluated on the LoCoMo benchmark for complex temporal and multi-hop reasoning.

Finding: Synapse significantly outperforms state-of-the-art retrieval methods on complex temporal and multi-hop reasoning tasks, specifically addressing the "Contextual Tunneling" problem where standard RAG fails to connect dispersed but related memories. The spreading-activation mechanism enables associative recall across disconnected memory fragments without requiring pre-computed similarity links, directly relevant to how episodic memory retrieval leverages temporal context and semantic associations.

Towards Efficient Large Language Model Serving: A Survey on System-Aware KV Cache Optimization
Jiang et al. (Feng Liu, University of Melbourne) · ACL 2026 Findings, arXiv:2607.08057 (9 Jul 2026) LLM-Memory KV-Networks

Approach: Systematizes the rapidly growing literature on KV cache optimization for LLM serving from a system-behavior perspective. Organizes existing efforts into three dimensions: execution and scheduling (temporal), placement and migration (spatial), and representation and retention (structural). Analyzes cross-behavior co-design affinity and behavior-objective links.

Finding: Provides the first unified taxonomy of KV cache optimization strategies, revealing that most work addresses only one dimension in isolation, leaving significant opportunities for cross-dimensional co-design. The structural dimension (how KV pairs are represented and which are retained) maps most directly to memory mechanisms — directly relevant to understanding how transformer memory systems can be made more efficient without sacrificing the ability to retain and retrieve associations across long contexts.

Understanding Axes of Difficulty For Long Context Tasks Via PredicateLongBench
Jain & Velingker (Google DeepMind) · arXiv:2607.08284 (9 Jul 2026) LLM-Memory

Approach: Proposes PredicateLongBench, a benchmark that stress-tests long-context reasoning by asking models to identify the longest contiguous subsequence satisfying given predicates (e.g., lexicographic ordering). Systematically explores multiple axes of difficulty including predicate complexity, sequence length, and distractor density. Provides both synthetic and real-world generation pipelines.

Finding: Frontier models struggle significantly as predicate complexity scales, demonstrating that current long-context capabilities are brittle — models can handle simple needle-in-haystack tasks but fail when predicates require structured reasoning over the full context. The benchmark reveals that context-window capacity alone is insufficient; structured attention over long sequences remains a fundamental bottleneck in how transformer memory is accessed.

S-EMBER: A Large-Scale Benchmark for Streaming Egocentric Memory Retrieval
Wang et al. (Wen-Tau Yih, Meta AI) · arXiv:2607.02689 (2 Jul 2026) LLM-Memory

Approach: Introduces S-EMBER, a large-scale benchmark for streaming episodic memory retrieval comprising 3,141 videos (388 hours) captured via Ray-Ban Meta smart glasses. Formalizes "grounded streaming episodic retrieval" — causal, active recall triggered by visual events in a continuous stream, with 9,448 QA pairs requiring precise temporal localization. Evaluates frontier VLMs and video-language models.

Finding: Uncovers a "localization paradox": semantic reasoning about past events improves with model scale, but temporal grounding precision remains stagnant regardless of parameter count, resolution, or frame density. This suggests a fundamental architectural bottleneck in how current models perform episodic memory retrieval — they can reason about what happened but not when, a dissociation with direct implications for models of temporal context in episodic memory.

Encoding-Retrieval
Learning regularities in noise engages both neural predictive activity and representational changes
Tirou et al. (Romain Quentin, INSERM Lyon) · Nature Communications, 7 Jul 2026 Encoding-Retrieval

Approach: MEG study tracking neural dynamics as human participants learned non-adjacent temporal dependencies embedded in visual-motor noise. Used multivariate pattern analysis (MVPA) to dissociate stimulus-specific predictive activity from representational similarity changes. Examined network-level contributions across sensorimotor, dorsal attentional, and frontoparietal systems.

Finding: Reveals two temporally dissociable learning mechanisms: (1) rapid neural predictive activity, with stimulus-specific patterns emerging before stimulus onset and preceding behavioral improvement; (2) slower representational change, characterized by gradually increasing neural pattern similarity between statistically dependent but non-adjacent elements. The temporal hierarchy — prediction first, then representational consolidation — maps onto the encoding-to-consolidation transition central to episodic memory formation. Sensorimotor and dorsal attention networks were the primary hubs for both processes.

When Does Continual Learning Require Learning
Harrington et al. (Trevor Darrell, Jitendra Malik, Yutong Bai — UC Berkeley / Meta) · arXiv:2607.07847 (8 Jul 2026) Peripheral

Approach: Reframes LLM continual learning as a problem of adapting to environmental change along two axes: space (new domains) and time (data drift under a fixed task). Compares prompt-based methods (GEPA, ACE), supervised learning (SFT, SDFT), reinforcement learning (GRPO, SDPO), and context compression (Cartridges, In-place TTT) under a unified evaluation protocol using recast LLM benchmarks.

Finding: No single method dominates: prompt-based methods adapt quickly but degrade on future tasks; distillation accumulates stably but struggles to update outdated facts; online RL handles knowledge drift best but is sensitive to noisy rewards. The core insight — that different patterns of environmental change require fundamentally different memory/update behaviors — echoes the complementary learning systems framework: fast hippocampal-like adaptation vs. slow cortical consolidation, with the optimal strategy depending on the statistics of change.

Hidden Decoding at Scale: Latent Computation Scaling for Large Language Models
Liu et al. (Miao Fan, WeLM / ByteDance) · arXiv:2607.08186 (9 Jul 2026) Peripheral

Approach: Proposes Hidden Decoding, a sequence-length scaling method that expands each token into n streams with independent embedding tables during continued pretraining. Keeps intermediate streams' KV cache as context, allowing more internal computation per token without widening Transformer layers. Introduces Stream-Factorized Attention to keep costs roughly linear in n. Validated at two scales: WeLM-HD4-80B and WeLM-HD4-617B (both at n=4).

Finding: Achieves first demonstrated sequence-length scaling at 100B+ MoE scale. Gains grow monotonically with expansion factor n, establishing that trading sequence length for per-token computation is a practical path for fixed-backbone scaling. The KV cache mechanism is central to the approach (intermediate streams' caches serve as extended context), making this a practical demonstration of how KV-based memory reuse enables more efficient computation at frontier scale.

⚠ Large team (48 authors), industrial-scale validation only (no academic-scale reproduction). Preprint; not yet peer-reviewed.
AutoPersonas: A Multi-Timescale Loop Engine for Open-Ended Persona Evolution
Mengchen Li · arXiv:2607.08252 (9 Jul 2026) Peripheral

Approach: Identifies "self-locking" as a failure mode in long-running persona agents where locally plausible events drive convergence toward stale behavioral patterns. Proposes an OSO (Occurrences → Observations → State) loop engine that separates divergent future-facing material from evidence-governed memory absorption. Tests on 8 models across 40-day simulations (1,600 events) plus a 3-year compressed run.

Finding: All direct-loop models crossed 90% action-category repetition by day 11; context-slice masking with per-sample divergence targeting reduced macro-theme repetition from 61.8% to 36.3% and roughly doubled cumulative theme count. The architecture's separation of memory consolidation from prediction-generation mirrors the complementary learning systems principle: evidence-governed absorption (hippocampus-like) vs. divergent state exploration (cortex-like), applied to agent identity maintenance.

⚠ Single-author preprint; simulation-based validation only, no real-world deployment.
Holographic Neural PCFG for Unsupervised Parsing
Yamaki et al. (Tadahiro Taniguchi, Ritsumeikan University) · arXiv:2607.08063 (9 Jul 2026) Peripheral

Approach: Applies Holographic Embeddings (circular correlation-based associative memory from knowledge graph literature) to PCFG rule scoring, giving every grammar rule probability a closed algebraic form. Uses torus-constrained embeddings for left-child, right-child, and lexical-emission relations. Evaluated on unsupervised constituency parsing across six languages.

Finding: Achieves state-of-the-art parsing performance while cutting rule-scoring parameters by 99.94% relative to neural baselines, demonstrating that algebraic associative memory structures can match or exceed black-box networks when the relational structure is well-matched to the task. The success of circular correlation as a binding mechanism for structured representations is noteworthy for models of compositional memory and relational binding.
