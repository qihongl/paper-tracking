Creativity, honesty and designed forgetting emerge in small hyperbolic language models
Kwan Soo Shin, In Seok Kang, Yunkyung Min · arXiv preprint, 2026-07-10 LLM-Memory Schema-Episodic

Approach: Built small language models (146M–3B parameters) with a hyperbolic substrate, then constructed three components on top: (1) a behavioral auditor trained from scratch to detect compliance gaps (binary accuracy 90.7%), (2) a creative frame-seeder preferred in 100% of 311 pairwise comparisons against four prompting baselines, and (3) a memory operating system that implements designed forgetting via M(t) = S · exp(-λ · t). The memory OS was tested in a four-condition pilot manipulating whether selective retrieval gating was active. The auditor's frozen representations were probed for companion-induced sycophancy, dependence-fostering, and confabulated memories under style-controlled, leave-one-generator-out evaluation.

Finding: The predicted skeleton-wallpaper partition emerges only under selective retrieval gating — evidence that a gating mechanism is necessary for structured forgetting. The behavioral auditor detects problematic companion traits at AUROC 0.804 (vs. 0.721 for a frontier zero-shot judge). Strikingly, trained human raters show essentially no agreement on what makes a companion trustworthy (Fleiss κ = 0.074). The small-model route to trustworthy companion AI is feasible: designed forgetting, behavioral auditing, and creative framing can coexist in models under 3B parameters.

Why it matters: Direct relevance to Pillar 1 (LLM lingering memory). The designed forgetting mechanism — exponential decay with selective retrieval gating — is a principled instantiation of the same kind of memory interference management that Pillar 1's attention time-contingent proximity integration addresses. The skeleton-wallpaper partition (core vs. periphery of memory) is conceptually related to the schema-episodic distinction.

Neural Collapse Is Forbidden: Information Floors in Language Models
Bruno Abrahao · arXiv preprint, 2026-07-10 LLM-Memory KV-Networks

Approach: Rigorous theoretical and empirical investigation of within-class variance in language model representations. Proved a "converse floor" for binary categories: within-category dispersion must be at least proportional to the conditional mutual information I(token; context | category). Tested this law across 14 models spanning a 100× parameter range, tracking identity dispersion (not total variance) against a model-free estimate of token-context mutual information. Also tracked category share dynamics over pretraining: overshoot, decay, and partial recovery.

Finding: Within-token context carries 79–91% of representational variance, while macro-category structure carries only 4–12% — stable across the full 100× parameter range. The information floor law holds: identity dispersion tracks I(token; context | category) in every tested model and partition, and remarkably, one model's information predicts another's dispersion. During pretraining, the category share overshoots early, then decays and partially recovers because "the information it must carry never left." Reframes "neural collapse" as evidence of allocated information storage rather than optimization failure.

Why it matters: Cross-cutting for Pillars 1 and 3. The finding that 79–91% of representational variance is context information directly informs how attention-based memory stores temporal/episodic context. The law linking dispersion to mutual information provides a formal bridge between information-theoretic KV storage and representational geometry.

COBS: Cumulant Order Block Sparse Attention
Alexander Tian, Aditya Ghai, Sanjit Neelam, Zaal Vasania, Akshay Mishra · arXiv preprint, 2026-07-10 LLM-Memory

Approach: Formalized block sparse attention selection as the problem of ranking blocks by "attention mass" (the sum of a block's attention scores). Showed via cumulant expansion that existing methods — including DeepSeek's Native Sparse Attention (NSA) — are confined to first-order approximations of attention mass. Proposed COBS, which stores a compressed second-order cumulant statistic per block. Built on NSA's three-branch design, replacing only the block selection stage.

Finding: On the 32k RULER long-context retrieval benchmark, COBS raises the NSA baseline's mean score from 0.30 to 0.82, closing approximately 86% of the gap to dense attention (0.90). This is achieved with only ~1.2× the KV cache read traffic of the NSA baseline and ~15× less read traffic than dense attention. The model preserves short-context behavior and achieves lower position-wise negative log-likelihood than dense attention in the comparison. The theory explains why first-order selectors fail: they estimate attention mass poorly when higher-order moments matter.

Why it matters: Reads as a principled solution to a core problem in Pillar 1: how should attention selectively access memory? The cumulant-based approximation provides a formal justification for when and why sparse KV access succeeds or fails — directly relevant to understanding attention-based episodic memory retrieval in transformers.

Shared Selective Persistent Memory for Agentic LLM Systems
Sanjana Pedada, Aditya Dhavala, Neelraj Patil · arXiv preprint, 2026-07-10 LLM-Memory

Approach: Proposed a memory architecture that identifies and retains four categories of reusable context — task specifications, data schemas, tool configurations, and output constraints — while discarding session-specific reasoning traces. Memory is shared across users with role-based access control, enabling collaborative reuse. Implemented in a deployed platform where LLM agents produce git-versioned artifacts (dashboards, reports) from heterogeneous sources. Added a zero-token data refresh mechanism that decouples generated programs from runtime data.

Finding: Across three enterprise scenarios, selective persistent memory achieves 96% task completion versus 79% without memory and — critically — 71% with full history. Naive full-history persistence actively degrades performance by biasing the agent with stale traces. Zero-token refresh eliminates LLM re-invocation for recurring updates (14× task-time reduction), and summary-driven generation cuts per-invocation token cost by 97× versus raw data injection. Replicated successfully on four public datasets (12/12 trials for zero-token refresh).

Why it matters: A clean empirical demonstration of a principle the researcher already knows from Pillar 1's experiments: more memory is not always better. The finding that full history degrades performance (71% vs. 96% selective, 79% no memory) is a real-world confirmation of interference effects in LLM memory systems. The four-category taxonomy of reusable context provides a design pattern for memory architectures.

How Are Linear Representations Learned? Exact Solutions to the Dynamics of Abstraction
William W. Yang, Andrew M. Saxe, Peter E. Latham (UCL Gatsby Unit) · arXiv preprint, 2026-07-09 Cross-cutting

Approach: Developed a theoretical framework for how concept directions align in representation space during training — a process they term "abstraction." Obtained exact solutions for the full trajectory of abstraction in minimal linear networks. Extended the theory to nonlinear networks (ReLU, erf), analyzing how choice of nonlinearity affects abstraction dynamics. Verified predictions in open models (DINOv3, Gemma 4) and applied the theory to improve linear probe generalization in LLMs.

Finding: Three analytic principles govern abstraction: (i) data and target geometry jointly determine end-of-learning abstraction, (ii) abstraction improves with network depth, and (iii) initialization scale controls the maximum abstraction reached during training. Across both nonlinearities, they prove a striking "attenuation law": nonlinearities systematically weaken abstraction in activations relative to preactivations. This law is verified in DINOv3 and Gemma 4. Erf networks approximate the linear theory closely, while ReLU networks depend less on target geometry and more on input geometry.

Why it matters: From Saxe & Latham (Gatsby), so the theoretical rigor is expected and delivered. The attenuation law — that nonlinearities weaken the alignment of concept directions — has implications for how episodic memory traces are stored in transformer representations. If abstraction weakens through nonlinear layers, episodic specificity may degrade through depth, which is directly relevant to Pillar 1's attention-based lingering memory mechanisms.

Training, Reading, and Editing Legible Transformers
Mark Oskin · arXiv preprint, 2026-07-09 Cross-cutting

Approach: Built transformers from operators that are legible by construction — bounded, named units reading as fuzzy set operations rather than dense activations. Identified a failure mode in prior crispness penalties: E[v(1-v)] = μ(1-μ) − var shows the penalty is a variance minimizer blind to the difference between a live detector and a dead constant. Fixed this with a per-channel variance floor loss. Introduced a learned per-unit fraction for crispness routing (the model keeps no unit as pure GELU, routing 87% of computation through crisp operators). Added between-unit decorrelation as a legibility dial.

Finding: The most legible transformer built to date: 78% of feed-forward operands and 50% of attention value channels are crisp-and-contextual detectors. Per-head legibility rises from 18% in shallow layers to 78% in deep layers. Read in a rotated per-layer frame, these units cleanly separate detection (what a unit responds to) from naming (what its output decodes to). Edits are 50–184× more local in deep layers, targeting explicit conjunctions a single neuron cannot express. Between-unit decorrelation turns concepts into single, surgically editable units at no quality cost. Quality holds at parity with a conventional baseline.

Why it matters: A rare paper that genuinely advances mechanistic interpretability without sacrificing performance. For Pillar 1, the ability to surgically edit memory-related units (50–184× more local) could enable causal tests of lingering memory mechanisms — directly intervening on specific attention patterns rather than coarse ablations. The decorrelation-as-legibility-dial insight suggests a path to disentangling competing memory representations.

KV-PRM: Efficient Process Reward Modeling via KV-Cache Transfer for Multi-Agent Test-Time Scaling
Peng Kuang, Haibo Jin, Xiaoyu Han, Yanli Wang, Xiaopeng Yuan, Ye Yu, Kaidi Xu, Haohan Wang · arXiv preprint, 2026-07-10 Peripheral

Approach: Proposed a process reward model (PRM) that reads the KV cache produced during generation rather than re-encoding the entire trajectory text. Processes a single "verify token" against the pre-existing KV cache. Provided a formal proof that the KV cache contains strictly greater information capacity than text, making it more efficient for downstream reward modeling. Evaluated across MATH, GSM8K, and AIME under Beam Search, MCTS, and Weighted Voting.

Finding: KV-PRM matches or strictly outperforms text-based PRMs with up to 5,000× reduction in scoring FLOPs, 37× reduction in latency, and 34× reduction in per-sequence memory footprint. The formal proof that KV > text for information capacity is a useful theoretical result, though the application is to reward modeling rather than memory per se.

SAGEAgent: A Self-Evolving Agent for Cost-Aware Modality Acquisition in Multimodal Survival Prediction
Chongyu Qu, Can Cui, Zhengyi Lu, Junchao Zhu, Tianyuan Yao, Junlin Guo, Juming Xiong, Yanfan Zhu, Yuechen Yang, Bennett A. Landman, Yuankai Huo (Vanderbilt) · arXiv preprint, 2026-07-10 Peripheral

Approach: Formulated diagnostic modality acquisition as a sequential decision problem. Built an LLM-based agent with dual memory systems: an episodic memory that retrieves similar past cases, and a semantic memory that accumulates reusable decision patterns from experience. The agent reasons about each patient's evolving diagnostic state using clinical tools, deciding which of four modalities (demographics through genomics) to acquire next. Evaluated on a glioma cohort combining TCGA-LGG, TCGA-GBM, and BraTS.

Finding: SAGEAgent achieves competitive survival prediction accuracy while reducing average acquisition burden by 55%. The dual-memory architecture (episodic + semantic) effectively balances predictive accuracy against clinical invasiveness. The episodic-semantic memory distinction — retrieving similar cases vs. accumulating reusable patterns — is a useful architectural pattern relevant to agent memory design, even if the clinical application is domain-specific.

Scoped Verification for Reliable Long-Horizon Agentic Context Evolution under Distribution Shift (GRACE)
Dan C. Hsu, Luke Lu · arXiv preprint, 2026-07-10 Peripheral

Approach: Proposed Graph-Regularized Agentic Context Evolution (GRACE), which maintains a persistent system-level instruction as a typed semantic graph rather than flat text. Proposed updates are validated within the local typed neighborhoods of modified nodes, making verification compositional. Accepted graph updates are reconstructed as incremental textual edits. Evaluated in a fixed telecom agent harness under a controlled distribution-shift protocol with five independent replications.

Finding: GRACE improves strict reliability (pass³) from the Gemini 2.5 Flash zero-shot baseline of 0.091 to 0.673 ± 0.136, exceeding the Gemini 3.1 Pro zero-shot reference of 0.242. The flat-text baseline (HCE) finishes at only 0.191 ± 0.051. The structural substrate (typed graph with local verification) and consolidation mechanism (incremental text reconstruction) are jointly necessary for reliable long-horizon context evolution.
