LLM-Memory
The Severance Problem: LLMs are Unaware of the Person Beyond the Prompt
Dor Litvak, Liu Leqi · arXiv cs.CL, 2026-07-15 LLM-Memory

Approach: Argues that current personal assistants fail (sycophancy, overconfidence, hallucination) because they lack an explicit representation of what remains unknown about the user. Proposes the “Severance Schema” — a structured list of dimensions along which the model lacks user knowledge (physicality, temporality, consequences, continuity, multiplicity, interiority) — injected into the context. Evaluated across five model families.

Finding: With the Severance Schema, assistants consistently reduce sycophancy, harmful advice, and hallucination, and switch from confidently extrapolating from incomplete user info to asking clarifying questions when something is missing. The mechanism — representing structured ignorance about the user — is directly relevant to how LLMs encode a “user memory” beyond the literal prompt, a close analogue of the lingering-user-state problem in attention-based memory.

Partition, Prompt, Aggregate: Statistical Self-Consistency in Language Models
Patrik Wolf, Thomas Kleine Buening, Andreas Krause, Celestine Mendler-Dünner · arXiv cs.CL, 2026-07-16 LLM-Memory

Approach: Treats in-context learning as conditional inference and tests whether LLM estimates satisfy the law of total probability: prior-weighted subpopulation conditionals should aggregate to the population marginal over any valid partition. Uses recursive binary-tree partitions of a population, verbalized subpopulation prompts, and compares aggregated estimates across partition granularities.

Finding: Across domains and frontier models, LLM estimates show widespread violations of basic probabilistic consistency. A “macro fallacy” emerges: estimates reconstructed from finer subpopulation responses align better with human reference data than direct population-level estimates, and the effect persists across tree structure and task. The authors argue models hold the relevant subpopulation knowledge but fail to propagate it into aggregates — establishing statistical self-consistency as a reference-free diagnostic of ICL reliability.

Memory as a Controlled Process: Learned Adaptive Memory Management for LLM Agents
Eric Hanchen Jiang, Zhi Zhang, …, Kai-Wei Chang, Ying Nian Wu · arXiv cs.CL, 2026-07-15 LLM-Memory

Approach: Frames memory operations for LLM agents as a Markov Decision Process and learns an online policy (a lightweight tabular contextual bandit with UCB exploration) that adaptively decides when/what/how much to retrieve, when to inject a distilled plan, and when to consolidate or forget. Backend-agnostic, learns from task-by-task binary feedback with no pretraining and no extra LLM calls; converges within tens of tasks.

Finding: Across 6 benchmarks, 3 agent frameworks, and 3 LLM backbones, the policy (“MemCon”) beats multiple memory baselines by up to 15.2 points in task success while cutting token use 5–20%. Core claim: optimal memory access is context-dependent (sparse early retrieval, plan reuse for recurring goals, re-retrieval when stuck) — a direct argument against fixed hand-designed retrieval heuristics, with clear bearing on gated/controlled attention-based memory access.

KV-Networks
PReM: Learning What to Preserve and When to Refresh for Context Compression
Bohan Yu, Lei Shen, Chenxi Zhou, …, Yu Cheng, Bo Zheng · arXiv cs.CL, 2026-07-15 KV-Networks

Approach: A context-compression framework that maintains the long context as the model’s internal layer-wise KV memory. A dedicated memory layer selects what to preserve, and a special <m> memory token triggers refreshes during generation. Trained with Phase-Separated Refresh Training, which aligns memory selection with memory-conditioned generation while preserving continuity across refreshes.

Finding: On 32K-token contexts, PReM outperforms strong compression baselines under both 16× and 32× compression while balancing answer quality and inference efficiency. The mechanism — keeping the compressed context adaptively refreshable as generation proceeds — speaks directly to KV memory networks and to how much contextual evidence stays accessible downstream, paralleling the position-preference/format confound studied in the lingering-memory experiments.

Smarter and Cheaper at Once: Byte-Exact KV-Cache Grafting Turns a Frozen Small Model into a Verified-Knowledge Flywheel
Sietse Schelpe · arXiv cs.CL / cs.LG, 2026-07-15 KV-Networks Cross-cutting

Approach: Deposits verified knowledge once as a byte-exact key-value (KV) state artifact and restores it by “graft” into a fresh inference context (no weight changes). The restore is bit-exact under a pinned deterministic configuration (SHA-256 equality, zero KL divergence, 100% argmax agreement over 50 samples); own-position graft is proven the unique numerically exact operating point on a model with floating-point rotary encoding.

Finding: On AIME 2025, a frozen Gemma-4-12B moves 80.0%→93.3% after grafting a verified solution library (above its 31B sibling’s 89.2% published anchor); the same byte-exact store widens usable context from 32,768 to 2,854,766 tokens at zero extra accelerator memory and moves bit-identically between machines. Frames KV states as addressable, transferable memory — a clean instantiation of KV memory networks as verified-knowledge stores.

⚠ Single-author; the engine is proprietary and every reported number is backed only by committed input/output hashes (one pre-registered replay claimed). Results are not independently reproducible from the paper alone.

Supervised Fine-Tuning vs. In-Context Learning: An Equilibrium Analysis of LLM Personalization under Congestion
Fengzhuo Zhang, Zhuoran Yang, Dirk Bergemann · arXiv stat.ML, 2026-07-15 Cross-cutting

Approach: A tractable framework for LLM serving that captures the statistical-economic trade-off between expensive supervised fine-tuning (SFT) and lightweight in-context learning (ICL) for personalization, under shared computational resources. Derives equilibrium resource consumption and validates predictions with GPT-2 on linear-regression tasks; cross-checks against a review of 21 major AI platforms.

Finding: ICL and SFT dominate in different regimes set by an interplay of pretraining coverage and data signal-to-noise ratio, but congestion from other users can flip the ranking. Equilibrium resource consumption is non-monotonic (better pretraining precision reduces congestion; broader coverage and harder tasks can increase it). The share of platforms offering both SFT and ICL rose from 9.5% (2021) to 71.4% (2025). Useful theoretical scaffolding for when in-context memory beats weight-based personalization.

Gate-Zero Growth: A Geometric Framework for Function-Preserving Continual Learning
Dante Lok · arXiv cs.LG, 2026-07-16 Cross-cutting

Approach: A function-preserving operator that adds new residual blocks through a zero-initialized gate. Under a transversality condition it induces “rank separation” in the functional Jacobian: old directions are unchanged, new-weight directions are exactly flat at the growth point, and new gate directions are the only first-order source of new functional variation. Analyzes function drift O(α²) and Jacobian leakage O(α∞).

Finding: On a 300M→857M Transformer adapted from WikiText-103 to BookCorpus, gate-zero growth reaches near-zero old-domain forgetting (ΔA<0.1) under both Isolation and Freeze-Nothing operating points, while a non-function-preserving control (Gstack) suffers roughly an order-of-magnitude larger forgetting under the same recipe. The same geometric analysis covers LoRA, ReZero, and zero-init adapters — establishing a canonical local geometry for safe capacity activation in continual learning (i.e., catastrophic forgetting).

xHC: Expanded Hyper-Connections
Xiangdong Zhang, Xiaohan Qin, …, Junchi Yan · arXiv cs.LG / cs.CL, 2026-07-16 Cross-cutting

Approach: Extends Hyper-Connections — which expand the transformer residual stream into N parallel streams, framed as “memory scaling beyond width and depth.” xHC adds temporal feature augmentation for richer write-back and a sparse residual-stream update (activates k=4 of N=16 streams while retaining dense access to the full residual state), scaling meaningfully beyond N=4 for the first time; xHC-Flash cuts per-sublayer memory traffic from 73.5C to 40C.

Finding: On 18B and 28B MoE models, xHC improves the average downstream score by +4.0 points over the prior mHC at only modest extra FLOPs; scaling-law experiments show vanilla and mHC require 1.50× and 1.19× the compute, respectively, to reach xHC’s loss. Frames residual-stream expansion as a memory-capacity scaling axis for transformers — an architectural lens on how much intermediate state a model can hold.

⚠ Technical report; reported gains are benchmark-average deltas, not direct memory-mechanism probes.

Thoughtseeds as Latent Causes: A Dual-Process Computational Phenomenology of Focused-Attention Meditation
Prakash Chandra Kavi, Daniel Ari Friedman, Gustavo Patow · arXiv q-bio.NC, 2026-07-16 Peripheral

Approach: A computational phenomenology of focused-attention meditation traversing four attractor states (breath focus, mind-wandering, meta-awareness, redirect attention). A dual-process active-inference model: L1 a stochastic Ornstein–Uhlenbeck process over attentional Yeo networks; L2 a generative model encoding latent “thoughtseeds” and autonomic action tendencies; L3 a metacognitive monitor implementing a Global Neuronal Workspace (GNW) capacity bottleneck that gates those tendencies.

Finding: Meta-awareness functions as the GNW ignition signal, derived from policy-prior divergence and competition between orchestrator and distractor thoughtseeds; policy selection minimizes expected free energy, and L2 actions furnish descending predictions that close the enactive perception–action cycle. Trained via variational EM on expert/novice phenotypes, simulations reproduce behavior “consistent with” contemplative-neuroscience findings — a tractable bridge between first-person phenomenology and neurophysiology, of peripheral interest as a computational cognitive model.

⚠ Simulation-only; “consistent with” empirical findings rather than a direct experimental test, and no new data.
