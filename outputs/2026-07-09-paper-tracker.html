Co-LMLM: Continuous-Query Limited Memory Language Models
Yair Feldman, Linxi Zhao, Nathan Godey, Dongyoung Go, Yilun Hua, Kilian Q. Weinberger, Jennifer J. Sun, Yoav Artzi (Cornell) · arXiv, 2026-07-08 LLM-Memory

Approach: Limited memory language models (LMLMs) externalize factual knowledge during pretraining to a knowledge base rather than memorizing it in weights. Co-LMLM departs from prior relational-KB designs by pairing continuous vector keys with textual knowledge values — the model generates flexible vector queries at minimal cost during generation to fetch attributional, human-readable knowledge. An annotation pipeline tags free-form factual spans in arbitrary text (not restricted to Wikipedia). Evaluated across pretraining on Wikipedia and FineWeb-Edu at multiple model scales (up to 360M).

Finding: Co-LMLM outperforms prior LMLMs and vanilla LLMs in both perplexity and factual precision. At 360M scale, it achieves lower perplexity than models pretrained on 40× more data (a roughly 40× data efficiency gain). SimpleQA-verified performance is in line with gpt-4o-mini and higher than Claude Sonnet 4.5. This demonstrates that decoupling factual storage from parametric memory can be done with continuous-key retrieval rather than rigid relational schemas, preserving both efficiency and attribution. Directly relevant to Pillar 1 questions about how LLMs should architect episodic vs. semantic memory stores.

When Does In-Context Search Help? A Sampling-Complexity Theory of Reflection-Driven Reasoning
Yotam Wolf, Noam Wies, Amnon Shashua (Hebrew University of Jerusalem) · arXiv, 2026-07-07 LLM-Memory Cross-cutting

Approach: Provides a theoretical analysis of in-context search — where LLMs iteratively generate, critique, and revise solution attempts — by modeling it as approximate inference over reasoning traces. The base model defines a prior distribution over traces, and self-reflection provides feedback for posterior updates. Studies the resulting sampling complexity: how many sequential attempts are needed to achieve high success probability. Also shows that the optimal policy under stagewise RL with verifiable rewards implements the same posterior reweighting rule. Key qualitative predictions are validated on real large reasoning models.

Finding: When reflections reliably localize early mistakes, in-context search yields exponential improvements over the base model — solving problems with exponentially small zero-shot pass rates using only a polynomial number of sequential attempts. When this localization property fails, conditioning on past attempts offers no asymptotic benefit over parallel sampling. The gains are learnable: cross-entropy training on search rollouts recovers the required behavior with polynomial sample complexity. This provides a formal framework for understanding when iterative in-context memory retrieval succeeds vs. fails, directly relevant to Pillar 1’s investigation of attention-mediated memory recall dynamics.

MILES: Modular Instruction Memory with Learnable Selection for Self-Improving LLM Reasoning
Ruilin Tong, Dong Gong (UNSW) · arXiv, 2026-07-08 LLM-Memory

Approach: MILES maintains modular memory units consisting of asymmetric pairs of sub-goal embeddings and sub-instructions, each with a learnable selection head. Operates in two stages: coarse-level memory expansion collects supervision for training selection heads from confident samples, while fine-stage learned selection reranks candidates for uncertain samples. Designed for test-time settings where memory expands incrementally and only limited supervision is available, avoiding the need for large-scale RL training data and fixed action spaces.

Finding: MILES consistently matches or outperforms prior memory-based reasoning methods while achieving superior accuracy-efficiency tradeoffs. The asymmetry between sub-goal embeddings (for retrieval) and sub-instructions (for execution) is architecturally notable, as it mirrors the cue-target distinction in human memory models. Demonstrates effectiveness, robustness, and transferability across diverse reasoning benchmarks. Relevant to Pillar 1 as an architectural exploration of how modular, learnable memory selection can improve sequential reasoning.

Sparse Delta Memory: Scaling the State of Linear RNNs through Sparsity
Loïc Cabannes, Pierre-Emmanuel Mazaré, Gergely Szilvasy, Matthijs Douze, Maria Lomeli, Ilze Amanda Auzina, Justin Carpentier, Gabriel Synnaeve, Hervé Jégou (Meta FAIR) · arXiv, 2026-07-08 LLM-Memory KV-Networks

Approach: Extends the Gated DeltaNet architecture by replacing the dense key-value outer product — the core memory update in linear attention — with sparse reads and writes to a large explicit memory. Uses a sparse addressing scheme to scale the hidden state to orders-of-magnitude higher capacity while maintaining isoFLOP constraints and identical parameter counts. Additionally, the initial state of the SDM memory is learned and used as a parametric memory, further improving performance on common-knowledge and reasoning tasks.

Finding: Under isoFLOP constraints, higher state memory capacity (achieved through sparse rather than dense KV operations) significantly improves performance on in-context learning and long-context retrieval tasks. The parametric memory initialization provides additional gains on knowledge and reasoning benchmarks. This directly demonstrates that memory capacity — not just parameter count or compute — is the bottleneck for ICL performance in linear recurrent models. Highly relevant to Pillar 1 (attention-based memory) and Pillar 3 (KV memory architectures).

The Hippocampus as a Generative Model
S. Nour, L. Gregorio, X. Peng, V. Messina, H. C. Barron (Oxford / MRC Brain Network Dynamics Unit) · Philosophical Transactions of the Royal Society B, 2026-07-09 Schema-Episodic Cross-cutting

Approach: A theoretical synthesis that recharacterizes the hippocampus as a generative model — a system that infers latent causes of sensory input and generates new data samples. Draws on evidence from hippocampal remapping (global, rate, structured), splitter cells, fear renewal, theta phase precession and sweeps, SWR replay (including never-experienced trajectories), targeted memory reactivation (TMR), and dopaminergic modulation. Extends the cognitive map view by formalizing hippocampal representations as inferred hidden states and proposing a dual online/offline mechanism: perceptual inference during theta vs. generative sampling and credit assignment during SWRs.

Finding: Proposes six testable predictions that bridge cellular, circuit, and behavioral levels: (1) late-theta sweeps reflect generative sampling of futures, (2) internally generated prediction errors during sleep update the model, (3) hippocampal dopamine is necessary for new latent state formation, (4) reverse replay back-propagates credit through the generative model, (5) mEC structural codes constrain replay to plausible trajectories, and (6) psychosis symptoms arise from impaired latent state segmentation. Directly relevant to Pillar 2’s schema-guided episodic memory modeling — the latent cause inference framework maps naturally onto schema formation and updating.

Recurrent Connectivity Shapes Spatial Coding in Hippocampal CA3 Subregions
Eunji Kong, Erfan Zabeh, Zhenrui Liao, Tiberiu S. Mihaila, Caroline Wilson, Charan Santhirasegaran, Darcy S. Peterka, Tristan Geiller, Attila Losonczy (Columbia University) · Neuron, 2026-07-08 Schema-Episodic

Approach: Using in vivo two-photon calcium imaging in mice during spatial navigation, combined with genetic perturbation of recurrent connectivity and computational modeling, the authors map how gradients in excitatory recurrent connectivity along the transverse axis of hippocampal CA3 (proximal vs. distal relative to DG) produce distinct spatial coding strategies. Specifically examines how recurrent collateral density shapes place field properties and population-level representations.

Finding: CA3 subregions with denser recurrent connectivity (proximal CA3, closer to DG) produce more pattern-completion-like dynamics — place fields are more stable under cue degradation but less discriminable between similar environments. Distal CA3, with sparser recurrence, shows sharper context discrimination at the cost of reduced stability. This demonstrates that the canonical CA3 recurrent collateral system is not uniform but graded, organizing a continuum from pattern completion to pattern separation along a single anatomical axis. Directly relevant to Pillar 2’s computational modeling of hippocampal pattern separation/completion dynamics.

KV-Networks
Fractal KV-Cache Archives: Lossless Symbolic Storage with In-Place Retrieval for Long-Context LLM Inference
Vladimir Gusev (independent) · arXiv, 2026-07-08 KV-Networks

Approach: Studies a complementary question to KV cache compression: once KV states are quantized to codebook indices, how should the resulting symbol stream be stored? Applies contractive iterated-map codes that serialize a symbol sequence into low-dimensional real vectors, forming a lossless archive format with O(1) random access and O(1) amortized append. A controlled study with GPT-2 at 1024-token contexts quantizes the archive, keeping a small exact window (4 attention sinks + 32 recent tokens) and archiving the rest via per-head residual vector quantization.

Finding: Achieves 36–54× compression relative to fp16 KV cache at a perplexity cost of 11–15%. Discovers a sharp key/value asymmetry: quantizing keys is roughly 4× more damaging than quantizing values, consistent with prior low-bit KV findings. This asymmetry is theoretically significant for Pillar 3 — it suggests that key vectors carry more fine-grained retrieval-relevant information than value vectors, potentially because keys encode positional/contextual address information while values encode content that is more compressible. The archive also functions as a search index, enabling approximate substring queries directly on stored vectors without materializing surrounding text.

KVpop: Key-Value Cache Compression with Predictive Online Pruning
Lukas Hauzenberger, Niklas Schmidinger, Anamaria-Roberta Hartl, David Stap, Thomas Schmied, Sebastian Böck, Günter Klambauer, Sepp Hochreiter (JKU Linz / NXAI) · arXiv, 2026-07-06 KV-Networks LLM-Memory

Approach: Learns a fixed-budget KV cache eviction policy by directly supervising the keep-or-drop decision against a future-attention target — computed efficiently without materializing dense attention maps. Introduces a delayed memory-based scorer that defers scoring for a fixed number of steps to exploit near-future context, uniquely among learned eviction methods. The scorer is trained on pre-computed generation traces using only key and value vectors.

Finding: On AIME and HMMT mathematical reasoning benchmarks, KVpop retains 98% of full-attention performance on Qwen3-4B at 75% KV cache compression, and 97% at 88% compression. Qwen3-8B shows even stronger results, reaching near-full teacher performance. The delayed memory-based scorer is the key architectural innovation — it demonstrates that exploiting near-future context for eviction decisions is more effective than immediate scoring. Relevant to Pillar 3 because it shows that memory retention decisions should be temporally extended, not instantaneous — echoing the temporal context dynamics studied in human free recall models.
