Editorial Note

Today's haul is dominated by LLM memory architecture — 6 of 10 papers from arXiv cs.CL/cs.AI address how transformers store, forget, retrieve, and manage memory. Two convergent themes stand out: (1) the proxy-behavior gap in memory evaluation, where perplexity improvements mask zero behavioral recall (Song et al.) and where deletion audits reveal that residual knowledge lives in the retrieval graph, not the parameters (Raeesi & Roed); and (2) memory as a learnable cognitive skill rather than a fixed architectural component, with AutoMem and Procedural Memory Distillation independently converging on training-time memory optimization. On the biological side, Julian et al. provide rare causal evidence — using optogenetic engram reactivation with simultaneous large-scale mPFC recordings — that hippocampal memory traces reconfigure prefrontal task representations in real time. Zeraati et al.'s Nature Neuroscience review on neural timescales offers a bridging framework: the τfit vs τparam distinction is directly applicable to comparing hippocampal replay timescales with transformer attention windows. Notably absent: no dedicated encoding-retrieval papers today, and no new KV/serial-position computational models.

LLM-Memory
Beyond Perplexity: A Behavioral Evaluation Framework for Deployment-Memory Claims in LLM Test-Time Training
Song, Chen, Kong, Xie, Dong, Chen & Zhang (Kun Zhang) · arXiv cs.CL, 1 Jul 2026 LLM-Memory

Approach: Introduces a behavioral evaluation framework that calibrates TTT memory claims to evidence strength. Two components: a claim-calibrated evidence ladder separating stream adaptation, bridge internalization, and deployment-time behavioral learning; and a diagnostic protocol with matched explicit-memory baselines. Validated by auditing recent TTT work and instantiating a controlled sparse nonce-fact experiment with one-step LoRA updates across three Qwen3 scales.

Finding: One-step LoRA updates successfully lower support and answer loss, but generated free-form recall stays at zero — exposing a measurable gap between proxy improvement and deployment behavior. This is the cleanest demonstration yet that perplexity and token-loss improvements do not constitute memory. The framework provides a concrete standard for authors to align claims with evidence, directly relevant to the lingering memory project's emphasis on behavioral memory metrics over engineering proxies.

Logit-Contribution Scoring Identifies Non-Literal Retrieval Heads
Gema, Alex & Minervini · arXiv cs.CL, 1 Jul 2026 LLM-Memory KV-Networks

Approach: Proposes LOCOS (Logit-Contribution Scoring), a write-aware detector that scores attention heads by projecting OV-circuit output onto the answer-token unembedding direction, contrasting needle and off-needle source positions in a single forward pass. Evaluated on the NoLiMa non-literal retrieval benchmark across Qwen3, Gemma-3, and OLMo-3.1 families.

Finding: Existing attention-based detectors miss non-literal retrieval heads because they reward literal copy (where attended token = generated token), ignoring what the head writes through its OV circuit. LOCOS captures write-path contributions: ablating 50 top LOCOS heads on Qwen3-8B drops ROUGE-L from 0.401 to 0.000, while the strongest baseline still retains 0.292. The selected heads are retrieval-specific — parametric recall and arithmetic stay at baseline. Critically, this demonstrates that non-literal retrieval depends on a sparse set of heads whose removal produces catastrophic failure, echoing the sparse-coding logic in KV memory networks.

Auditing Forgetting in Limited Memory Language Models
Raeesi & Roed · arXiv cs.CL, 1 Jul 2026 LLM-Memory

Approach: Proposes a causal auditing framework for LMLMs that holds the model fixed and varies the database state at inference time across three interventions (FULL, DEL-ON, DEL-OFF), decomposing post-deletion behavior into parametric leakage L(f), retrieval-mediated correctness R(f), and retrieval artifact rate. Applied to 12,228 alias-closure deletions across 13 databases with four adversarial topologies (Base, Alias, Noise, Collision) and six prompt formulations.

Finding: Parametric leakage is near zero in every variant and prompt style — the model rarely returns the deleted answer without retrieval. Residual correctness (0.7% on released LMLM database, up to 13.6% on most adversarial variant) is predominantly reconstituted from near-neighbor retrieval. Prompt formulation does not independently control how much of a deleted fact survives. The unlearning boundary is drawn by the database administrator, not the model — a clean separation that complicates claims about "memory" in retrieval-augmented systems.

AutoMem: Automated Learning of Memory as a Cognitive Skill
Wu, Zhu, Zhang, Wang & Yeung-Levy · arXiv cs.AI, 1 Jul 2026 LLM-Memory Cross-cutting

Approach: Treats memory management as a trainable cognitive skill (metamemory) rather than a fixed architectural component. Two-loop optimization: a strong LLM reviews complete agent trajectories and iteratively revises the memory structure (prompts, file schemas, action vocabulary), while the agent's own good memory decisions are identified from many episodes as training signal to sharpen memory proficiency directly. Evaluated on Crafter, MiniHack, and NetHack.

Finding: Optimizing memory alone — without modifying task-action behavior — improved base agent performance ~2–4×, bringing a 32B open-weight model competitive with Claude Opus 4.5 and Gemini 3.1 Pro Thinking. Memory management is an independently learnable skill separable from task performance. This aligns with the cognitive science metamemory literature — knowing what to encode and when to retrieve — and suggests a path toward learned rather than engineered attention-gating in lingering memory architectures.

KV-Networks
MosaicKV: Serving Long-Context LLM with Dynamic Two-D KV Cache Compression
Qiang, Chen, Wu, Gu, Hua, Xia, Zang & Chen (Haibo Chen) · arXiv cs.LG, 1 Jul 2026 KV-Networks

Approach: Dynamic two-dimensional KV cache compression exploiting non-uniform importance distribution within KV vectors. Instead of one global compression pattern, MosaicKV identifies important elements per KV vector and selects compression strategies at KV cache segment granularity. Introduces compressed KV cache management using underutilized GPU/CPU resources.

Finding: On H800 GPUs: up to 16× attention speedup, 4.8× lower decode latency, 7.3× higher throughput, 3× memory reduction, with only 1.76% average accuracy loss on LongBench and RULER. The key mechanistic insight: importance is non-uniform in both sequence and channel dimensions, so two-D compression outperforms either axis alone. Relevant to KV memory networks: suggests that not all key-value pairs in attention memory are equally informative — a sparse-coding principle that parallels hippocampal pattern separation.

Hippocampal Engrams Configure Prefrontal Context Representations to Guide Flexible Decisions
Julian, Kaminsky, Tank & Brody · bioRxiv, 6 Jul 2026 Schema-Episodic Cross-cutting

Approach: Combines tagging and optogenetic reactivation of hippocampal engrams with simultaneous large-scale recordings in medial prefrontal cortex (mPFC) during a context-dependent task-switching paradigm in mice. Tests whether HPC engrams can drive retrieval of a set of stimulus-response mappings (decision rules) rather than a specific motor response.

Finding: HPC engram reactivation caused mice to apply engram-consistent decision rules rather than a specific motor output — ruling out a simple stimulus-response account. Simultaneous mPFC recordings revealed that reactivating the HPC engram reinstated the context representation in mPFC within hundreds of milliseconds, mediated by rapid network effects. Other aspects of endogenous mPFC dynamics were left intact. This provides direct causal evidence that hippocampal memory traces can reconfigure cortical task representations in real time — a mechanism directly relevant to the Schema-Guided Episodic Memory RNN's core claim that schema-binding attention gates hippocampal-cortical interaction.

Neural Timescales from a Computational Perspective
Zeraati, Levina, Macke & Gao · Nature Neuroscience, 2 Jul 2026 Cross-cutting Schema-Episodic

Approach: Nature Neuroscience review synthesizing three computational directions: (1) how different data analysis methods quantify timescales across behavioral states and recording modalities; (2) how biophysical models — synaptic filtering, adaptation, recurrent connectivity, clustered connections, neuromodulation, criticality — mechanistically generate diverse timescales; and (3) how task-performing networks and ML models reveal the functional relevance of neural timescales. Introduces the τfit (data-estimated AC time constant) vs τparam (model internal parameter) distinction as a key comparative framework.

Finding: The review establishes τfit as a comparable fingerprint across biological and artificial systems — linking hippocampal time constants, cortical hierarchies, and RNN/transformer memory windows into a unified framework. Attractor dynamics represent the extreme of long timescales supporting working memory; task-optimized RNNs spontaneously develop timescale hierarchies mirroring biology. Directly applicable to all three pillars: provides a principled language for comparing hippocampal replay timescales (Pillar 2), KV cache decay rates (Pillar 3), and attention window effective memory spans (Pillar 1).

NeuroCogMap Reveals Cognitive Organization of Large Language Models
Sun, Lu, Ma, Li, Wang, Pang, Liu, Li, Sun, Wang, Zeng, Xu, Li & Wen · arXiv q-bio.NC, 1 Jul 2026 Cross-cutting

Approach: Cognitive neuroscience-inspired framework that organizes LLM internal features into functional parcels linked to interpretable functions, cognitive capabilities, and a cognitive hierarchy. Maps LLM failures (hallucination, bias, refusal failure, sycophancy) onto distinct disruptions in representational and behavioral-control systems. Also tests whether LLM functional organization predicts human cortical responses during naturalistic language comprehension.

Finding: LLM functional parcels form a stable, semantically coherent organization partly conserved across models and functionally linked to outputs. Major LLM failures correspond to distinct representational disruptions with internal signatures for mechanism-guided detection. Improves prediction of human cortical responses, with strongest correspondence in higher-order association cortex. The brain-to-LLM mapping framework could inform how hippocampal-like organizational principles in transformers relate to biological memory systems — an underexplored direction for the cross-cutting pillar.

Procedural Memory Distillation: Online Reflection for Self-Improving Language Models
Liu, Bansal, Pang, Li, Liu, Ming, Ke, Joty & Yavuz · arXiv cs.AI, 1 Jul 2026 Cross-cutting

Approach: Converts cross-episode signals from RLVR rollouts into reusable procedural memory organized at three levels: raw trajectories, self-reflected strategies/lessons, and higher-level behavioral patterns. A memory-conditioned self-teacher draws on accumulated experience to supervise the student, enabling progressive internalization of procedural knowledge. Key design: co-evolution — policy generates rollouts that update memory, and memory shapes supervision that updates policy.

Finding: PMD improves over SDPO by 3.8–5.5% on SCIKNOWEVAL and 7.9–13.6% on LIVECODEBENCH across Qwen3-8B and OLMo3-Instruct-7B. Co-evolution is essential: freezing either memory or policy trails PMD by >10%. The three-level memory hierarchy (trajectories → strategies → patterns) mirrors the abstraction ladder in schema-guided episodic memory — raw episodes, schema-extracted rules, and meta-cognitive patterns — suggesting convergence between LLM procedural memory and biological systems consolidation.

Peripheral
From Signals to Structure: How Memory Architecture Drives Language Emergence in LLM Agents
Talebirad, Redman, Parsaee & Zaiane · arXiv cs.AI, 30 Jun 2026 Peripheral

Approach: Lewis signaling game with two LLM agents inventing a shared language from scratch. Compares five memory architectures (stateless, rolling window, persistent private notebook, etc.) across varying channel capacities. Uses information bottleneck-inspired arguments to predict optimal capacity.

Finding: Memory architecture matters more than channel capacity for coordination success. Agents with persistent private notebooks achieve best coordination (0.867 ± 0.023 at capacity = 25), while stateless agents peak at moderate capacity then degrade — the notebook externalizes learned conventions. Channel capacity alone cannot predict coordination success. Adjacent to the lingering memory question: externalized memory stabilizes conventions across extended interactions, a form of schema-guided coordination.
