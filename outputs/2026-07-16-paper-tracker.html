LLM-Memory
Extending LLM Context via Associative Recurrent Memory
Gleb Kuzmin, Ivan Rodkin, Aydar Bulatov, Yuri Kuratov, Lyudmila Rvanova, Mikhail Katkov, Ilia Sochenkov, Misha Tsodyks, Timothy Baldwin, Mikhail Burtsev, Artem Shelmanov · arXiv, 2026-07-13 LLM-Memory KV-Networks

Approach: The authors investigate the Associative Recurrent Memory Transformer (ARMT) as a practical long-context solution for LLMs. They construct two domain-specific long-context datasets and propose a comprehensive training recipe combining continued pre-training, synthetic long-context data generation, curriculum learning, and selective associative memory integration into chosen model layers. The ARMT replaces standard quadratic self-attention with constant-memory recurrent associative memory, evaluated against in-limit transformer baselines.

Finding: ARMT-augmented models process inputs well beyond original context limits without degrading performance, generalize better to out-of-distribution context lengths, and require 30% fewer FLOPs while preserving baseline performance within the original context window. Note: Misha Tsodyks (Weizmann Institute) is a co-author — known for the mathematical theory of synaptic plasticity and working memory. This is a rare case of a computational neuroscientist contributing directly to LLM memory architecture, making this paper especially relevant to the researcher's bridge between computational memory models and transformer memory mechanisms.

Track, Rank, Crack: Epistemic Working Memory Scales Multi-Hop Reasoning in Language Agents
Ning Liu · arXiv, 2026-07-14 LLM-Memory

Approach: The author introduces SLEUTH, a framework that gives LLM agents an explicit structured epistemic working memory with three components: Confirmed Facts (source-grounded), Active Hypotheses (evidence-ranked), and Open Questions (action-driving). The agent maintains this state across multi-hop reasoning chains, rather than relying on implicit context-window memory. Tested across five multi-hop benchmarks against five established baselines (including Reflexion).

Finding: SLEUTH's advantage grows with difficulty: +5 points on HotpotQA, +11 on 4-hop chains, surpassing Reflexion without multiple episodes. A lightweight commitment trigger only works when the agent already maintains structured epistemic state — isolating organized state as the necessary condition. Enforcing protocol adherence on a weaker model recovers up to +19 points on the hardest problems, demonstrating that reasoning organization, not raw model capability, is the active ingredient for scaling multi-hop reasoning.

Speculate with Memory: Lossless Acceleration for LLM Agents
Yu Li, Qinyuan Ye, Prafulla Kumar Choubey, Jiaxin Zhang, Chien-Sheng Wu (Salesforce AI) · arXiv, 2026-07-14 LLM-Memory

Approach: Existing speculators for LLM agents are stateless — they discard all information between tasks, preventing prediction quality from improving with experience. The authors equip the speculator with three online memory systems that learn from past agent trajectories: (1) a contrastive transition table tracking action-sequence statistics, (2) an episodic memory retrieving contextually similar segments, and (3) a confusion tracker suppressing recurring errors. Evaluated on six benchmarks spanning action, observation, and chained prediction.

Finding: Memory-augmented speculation yields 19–39% relative accuracy improvement on action prediction and up to 2.5× increase on observation prediction tasks with repetitive action spaces. Gains grow continuously as memory accumulates and generalize across speculator models of varying cost. All speculation is lossless — it runs during idle time at zero added wall-clock cost.

Knowledgeless Language Models: Suppressing Parametric Recall for Evidence-Grounded Language Modeling
Roi Cohen, Yvan Carré, Nick Lechtenbörger, Hendrik Droste, Lucas Kerschke, Russa Biswas, Gerard de Melo, Jan Buys (HPI / U. Cape Town) · arXiv, 2026-07-14 LLM-Memory

Approach: The authors introduce Knowledge-"Less" Language Models (KLLMs), pretrained on corpora where named entities are anonymized to remove entity-linked factual supervision. This epistemic intervention is tested across multiple model scales on contextual QA, fact verification, hallucination detection, and retrieval-grounded benchmarks with imperfect evidence.

Finding: KLLMs substantially reduce closed-book factual recall while improving contextual task performance. In retrieval-grounded settings with imperfect evidence, KLLMs achieve 20–25% relative gains over standard LMs. They also exhibit better calibration (improved ECE, Brier score, AUROC) and more reliable abstention behavior. The key insight: suppressing entity-linked supervision during pretraining induces a shift from parametric recall toward evidence-grounded reasoning — a demonstration that pretraining-time control over knowledge acquisition directly shapes epistemic behavior.

Schema-Episodic
Differentiable Clone-Structured Causal Graphs for End-to-End Cognitive Map Learning from Image Sequences
Arash Nikzad, Sasan Sarbishegi, Ali Dasmeh, Muhammad Asif, Parsa Gharavi, Erik Husom, Sagar Sen, Andrew B. Lehr, Olivier Penacchio, Ana Clemente, Tristan M. Stöber · arXiv, 2026-07-14 Schema-Episodic Cross-cutting

Approach: The Clone-Structured Causal Graph (CSCG) is a normative hippocampus model that learns interpretable cognitive maps from aliased observations, but it requires predefined discrete alphabets and cannot be combined with neural network modules. The authors reformulate CSCG as a fully differentiable module (gradCSCG) and couple it to a VQ-VAE perceptual front-end. A soft emission forward pass allows the map-learning objective to flow back into perception, with loss-balancing mechanisms to prevent module collapse during joint training. Tested on symbolic grid worlds and MNIST image sequences across four heavily aliased environments.

Finding: Gradient-trained gradCSCG reproduces original CSCG results on symbolic grid worlds. On MNIST image sequences — where each visit to a location yields a randomly sampled digit image — the end-to-end pipeline successfully uncovers the underlying adjacency graph with high edge precision and recall. This provides proof-of-principle that a hippocampus-inspired cognitive map model can serve as a composable building block in deep learning architectures, bridging the gap between normative models of hippocampal computation and modern representation learning.

⚠ Preprint; no biological validation against neural data — purely a computational proof-of-concept. The VQ-VAE front-end is a simplification of real sensory processing.
KV-Networks
How Query Visibility Changes KV-Cache Compression Rankings: A Matched-Budget Audit
Daming Luo, Christy Liang, Junyu Xuan · arXiv, 2026-07-11 KV-Networks

Approach: Most KV-cache compression methods are evaluated with the query appended before compression (query-aware protocol). But the economic case for compressed KV cache is reuse: compress once, answer many queries. The authors conduct a matched-budget audit of six published compression methods against three trivial baselines on three open 7–9B models, with 144,300 paired evaluations on RULER-8192 and 40,800 on LongBench — all under both query-aware and query-agnostic protocols. Everything is held fixed except the scoring rule.

Finding: Query visibility changes the rankings dramatically. Under the agnostic protocol, only KeyDiff beats a best-of-3 trivial baseline consistently (31 of 36 cells). SnapKV — the most widely deployed method — loses to "keep the start and the recent window" on average (Δ = −0.066). The per-method drop between protocols is ordered consistently with how visible the query is to each method's scoring signal, legible in source code: SnapKV drops Δ = +0.198 (query sits in its 64-token observation window) while KeyDiff drops only Δ = +0.011 (its score contains no query term). This is a sobering audit that every KV-cache paper should be required to pass.

Forgetful Attention: A Trainable Support-Vector Memory with Certified Selection and Exact Unlearning
Vishwajith Ramesh · arXiv, 2026-07-13 KV-Networks LLM-Memory

Approach: Attention is reframed as an online learner over context, with weights derived from a one-class SVM with fixed box parameter C. The active-set partition gives reserve tokens exactly zero weight, certifying output-preserving eviction. A reversible incremental solver deletes a token to recover the state produced by retraining without it. Tested on MIMIC-IV medical streams for rare-item recall and deterioration tracking, enwik8 language modeling, and TinyStories.

Finding: Certified selection achieves 0.86 vs. 0.32 rare-item recall and retains 0.80 vs. 0.05 deterioration hours on MIMIC-IV streams. On enwik8, the SV-Attention hybrid obtains 2.178 BPC vs. 2.383 for a matched-state sliding-window Transformer across seven seeds (8.6% paired improvement, p = 0.001). A reversible solver recovers identical partitions under unique optima with median decision-function deviation ~10⁻⁹. Surgical forgetting, exact editing, and patient-record deletion are demonstrated as immediate applications.

A JoLT for the KV Cache: Near-Lossless KV Cache Compression via Joint Tucker and JL-Residual Allocation for LLMs
Rahul Krishnan, Volker Schulz · arXiv, 2026-07-14 KV-Networks

Approach: The authors observe that the KV cache is naturally a third-order tensor (heads × tokens × features), with very different redundancy across axes. JoLT applies a partial Tucker decomposition that compresses only the token and feature axes while leaving head and layer axes intact, then restores discarded energy with a Johnson-Lindenstrauss rotated low-bit residual. A single Lagrangian dual allocates Tucker ranks and residual bit-widths jointly across layer groups and separately for keys and values under a one-byte budget.

Finding: Near-lossless 2–3× compression: perplexity, GSM8K accuracy, and RULER needle-in-a-haystack retrieval all stay within statistical noise of the uncompressed baseline on Mistral-7B (GQA) and LLaMA-2-13B (MHA). At 2×, JoLT reconstructs the cache to relative Frobenius error 0.009 (K) and 0.006 (V) — roughly an order of magnitude below cross-layer SVD and 4-bit quantization. A randomized-SVD variant (FlashJoLT) delivers 5–13× compression-time speedup at matched quality.

Encoding-Retrieval
Comparing Semantic Navigation in Humans and Large Language Models using Natural Language Processing
Gabriel Paris-Colombo, Rodrigo M. Cabral-Carvalho, Felipe D. Toro-Hernández · arXiv, 2026-07-13 (CogSci 2026) Encoding-Retrieval Cross-cutting

Approach: Semantic memory retrieval is conceptualized as navigation through conceptual space. The authors compare retrieval dynamics between 82 human participants and three LLMs (GPT-4o, Gemini-2.5-Pro, Claude-Sonnet-4.5) across eight temperature settings using verbal fluency data. Three trajectory-based NLP metrics quantify complementary dimensions: entropy (step-size predictability), distance to next (successive semantic steps), and distance to centroid (global dispersion).

Finding: Humans exhibited higher entropy, larger semantic steps, and broader dispersion than all LLMs — indicating more variable and exploratory search. Temperature tuning produced only partial alignments: individual metrics matched at specific settings, but no configuration reproduced the complete human retrieval profile across all three dimensions. This suggests that human semantic search implements a distinctive balance between local exploitation and global exploration that current transformer architectures fail to reproduce — a finding relevant to the researcher's work on memory retrieval mechanisms and the gap between human and LLM episodic/semantic dynamics.

Cross-cutting
Constructed Reality, Contested Priors: Decoupling and the Architecture of Cognitive Relapse Under the Free Energy Principle
MD Ibrahim Hossain Ridoy · arXiv, 2026-07-12 (q-bio.NC cross-list) Cross-cutting Schema-Episodic

Approach: Under the free energy principle, a predictive system does not observe reality directly but maintains a generative model whose best hypothesis constitutes experience. The author asks: can a system undergo "ontological inversion" — permanently adopting a synthetic environment as its default? Studied via a convolutional VAE with recurrent latent predictor (ELBO ≡ variational free energy), trained first on a baseline domain, then on a mixed stream with a swept rehearsal ratio r controlling how much baseline content persists during transition to a target domain. Representational capacity (discrimination) is tracked separately from default behavior (unconstrained generation) across 90 runs.

Finding: Representational accuracy stays near ceiling (0.97–0.998) regardless of r, while default behavior spans nearly the system's entire range — a sharp decoupling of learning from acceptance. At intermediate r, the system's default output rises toward the target domain, then partially reverts toward baseline while training continues unchanged — a structural failure termed "cognitive relapse." This has provocative implications for understanding memory consolidation, schema updating, and why systems (biological and artificial) resist incorporating new priors even when they demonstrably "know" the new information.

⚠ Single-author preprint; the VAE-as-FEP mapping is a modeling choice, not an empirical demonstration. The term "cognitive relapse" is the author's coinage and has not been validated against biological data.
Peripheral
Sparse Inter-Layer Dependencies of Transformer FFN Neurons
Johannes Knittel, Hanspeter Pfister (Harvard) · arXiv, 2026-07-13 Peripheral

Approach: The authors introduce a training-free attribution method that estimates the relative influence of upstream FFN neurons and attention outputs on a target neuron's activation. They examine whether FFN neuron activations can be explained by a sparse set of preceding activations, using neuron-specific masking and propagation through the full network.

Finding: Small subsets of preceding activations suffice to preserve neuron outputs with high fidelity. When neuron-specific masks are applied simultaneously across all layers — such that induced deviations propagate through the network — model perplexity remains largely unchanged at moderate sparsity levels. This demonstrates that despite dense parameterization, FFNs exhibit sparse and structured inter-layer dependencies at the neuron level, with implications for circuit-level interpretability and efficient inference.
